# チーム9
チーム TAIYO


# ゲームタイトル
めを　さがせ！


# ゲーム概要
「め」を探してクリック！


# プラットフォーム
WEB (Scratch)


# ゲーム内容
「あ」と「め」が画面上に表示されるので、「め」をクリックすると次のレベルに進みます。  
レベルが進むと、「あ」が増えていき、難しくなります。  
頑張って「め」を探しましょう！  
レベル１２から、救済機能があります。  
どうしても「め」が見つからなかったら、スペースキーを押してみましょう！  
以下URLでプレイ及び、ソースコードの参照ができます。  
[https://scratch.mit.edu/projects/166371876/#player](https://scratch.mit.edu/projects/166371876/#player)
